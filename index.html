For GitHub Pages, place data.csv in your repo root — no upload needed.

CSV FORMAT — ONE ROW PER MEMBER

Each person is a single row. Add a new week_N column each week.

name,start_weight,goal_weight,week_1,week_2,week_3,week_4
Alex,230,190,228,225.5,223,220
Jordan,190,155,188,185,183.5,181
Sam,275,220,273,270,267.5,265
Casey,210,170,208,206,203,201

name — display name  |  start_weight — weight before challenge (lbs)  |  goal_weight — target (lbs)
week_1, week_2 … — recorded weight each week. Leave a cell blank if someone missed a weigh-in.

GITHUB PAGES DEPLOYMENT

1. Put index.html and data.csv in the root of your GitHub repo.
2. Go to Settings → Pages → Branch: main, folder: / (root) → Save.
3. Each week: open data.csv, add a new week_N column with everyone's weights, commit & push.
4. Site auto-updates in ~60 seconds. 🎉
